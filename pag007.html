Decisões de design

Principais aspectos de design do projeto Samples

    O site utiliza layout híbrido, o que contribui para o ocultamento das áreas vazias do site e um melhor aproveitamento de tela

    O site utiliza flexbox para uma melhor organização dinâmica dos elementos da página

    Os áudios mostram a transição da música original para a música que utiliza o sample (demonstrando os cortes, repetições, novos arranjos, maior ou menor tom/bpm, etc).
